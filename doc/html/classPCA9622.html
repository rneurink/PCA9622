◆ setGroupFrequency()

uint16_t PCA9622::setGroupFrequency ( uint16_t  ms,
EAddressType  addressType = EAddressType::Normal 
)

Sets the blinking frequency When DMBLNK is 0 this register does nothing When DMBLNK is 1 this register is used to program the blinking delay.

Parameters
msthe blinking delay in ms from 42..10666. Not all values are valid and it will find the closest valid value.
addressTypethe I2C address type to write to
Returns
uint16_t The real time in ms that the register will be programmed to as not all values are valid

◆ setGroupPWM()

void PCA9622::setGroupPWM ( uint8_t  value,
EAddressType  addressType = EAddressType::Normal 
)

Sets the group duty cycle. When DMBLNK is set to 0, a 190Hz fixed frequency signal is superimposed with the 97kHz individual brightness control signal. When DMBLNK is set to 1, togheter with the group frequency defines a group blinking pattern in which the value will determine the duty cycle of the ON/OFF ratio as a percentage.

Parameters
valueThe pwm duty cycle
addressTypethe I2C address type to write to
